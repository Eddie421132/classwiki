## 我该如何编辑这份代码？

你可以通过多种方式来编辑你的应用。

**使用 Lovable**

只需访问 [Lovable 项目](https://lovable.dev/projects/REPLACE_WITH_PROJECT_ID) 并开始输入提示即可。

通过 Lovable 所做的更改会自动提交到这个仓库。

**使用你偏好的 IDE**

如果你想在本地使用自己的 IDE 进行开发，可以克隆这个仓库并推送更改。推送的更改同样会反映到 Lovable 中。

唯一的要求是已安装 Node.js 和 npm —— [使用 nvm 安装](https://github.com/nvm-sh/nvm#installing-and-updating)

请按照以下步骤操作：

```sh
# 步骤 1：使用项目的 Git URL 克隆仓库
git clone <YOUR_GIT_URL>

# 步骤 2：进入项目目录
cd <YOUR_PROJECT_NAME>

# 步骤 3：安装所需的依赖
npm i

# 步骤 4：启动带有自动重载和即时预览的开发服务器
npm run dev
